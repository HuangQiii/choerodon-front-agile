# Agile Front
`Agile Front` is the core front service of Choerodon. The service is responsible for Agile process management and providing users with a better user experience through rich display.

## Features
- **Version Management**
- **Sprint Management**
- **Issues Management** 
- **Backlog Management**
- **Component Management**
